# Пресс-центр - Компонент новостей

## Реализованная функциональность

### Страницы
1. **Главная страница** (`index.html`)
   - Динамический блок с последними 5 новостями
   - Загрузка данных из Supabase
   - Ссылки ведут на детальные страницы статей

2. **Список новостей** (`news.html`)
   - Все опубликованные новости
   - Фильтрация по категориям:
     - Все
     - Новости
     - СМИ о Фонде
     - События Фонда
   - Пагинация с кнопкой "Показать ещё"
   - Адаптивная верстка

3. **Детальная страница** (`article.html`)
   - Полный текст статьи
   - Хлебные крошки для навигации
   - Изображение и метаданные
   - Кнопка возврата к списку новостей

### База данных

**Таблица:** `news_articles`

**Поля:**
- `id` (uuid) - уникальный идентификатор
- `title` (text) - заголовок статьи
- `slug` (text, unique) - URL-friendly идентификатор
- `excerpt` (text) - краткое описание
- `content` (text) - полный текст статьи
- `image_url` (text) - ссылка на изображение
- `category` (text) - категория статьи
- `published_at` (timestamptz) - дата публикации
- `is_featured` (boolean) - важная статья
- `created_at` (timestamptz) - дата создания
- `updated_at` (timestamptz) - дата обновления

**Категории:**
- `news` - Новости
- `media` - СМИ о Фонде
- `events` - События Фонда

**Security:**
- Row Level Security (RLS) включен
- Публичный доступ только на чтение
- Аутентифицированные пользователи могут создавать/редактировать

### Модули JavaScript

1. **src/supabase.js** - инициализация клиента Supabase
2. **src/news.js** - функции для работы с новостями
3. **src/home-news.js** - загрузка новостей на главную
4. **src/news-list.js** - список новостей с фильтрацией
5. **src/article-detail.js** - детальная страница статьи
6. **src/main.js** - главный модуль приложения

### Навигация

```
index.html (Главная)
├── Блок "Пресс-центр"
│   ├── Новость 1 → article.html?slug=...
│   ├── Новость 2 → article.html?slug=...
│   └── ...
├── Кнопка "Перейти в пресс-центр" → news.html
└── Меню → Пресс-центр → news.html

news.html (Список новостей)
├── Фильтры по категориям
├── Новость 1 → article.html?slug=...
├── Новость 2 → article.html?slug=...
└── Кнопка "Показать ещё"

article.html (Детальная страница)
├── Хлебные крошки
│   ├── Главная → index.html
│   └── Пресс-центр → news.html
├── Содержание статьи
└── Кнопка "Вернуться к новостям" → news.html
```

## Использование

### Получение списка новостей
```javascript
import { getNewsList } from './news.js'

const articles = await getNewsList({
  category: 'news',  // опционально
  limit: 10,         // опционально
  offset: 0,         // опционально
  featured: false    // опционально
})
```

### Получение детальной статьи
```javascript
import { getNewsArticle } from './news.js'

const article = await getNewsArticle('article-slug')
```

### Форматирование даты
```javascript
import { formatDate } from './news.js'

const formattedDate = formatDate('2025-03-15')
// Результат: "15 марта 2025 г."
```

### Получение названия категории
```javascript
import { getCategoryName } from './news.js'

const categoryName = getCategoryName('news')
// Результат: "Новости"
```

## Разработка

Запуск dev-сервера:
```bash
npm run dev
```

Сборка для production:
```bash
npm run build
```
