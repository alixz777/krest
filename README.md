# Фонд помощи "Русский Крест"

Проект веб-сайта благотворительного фонда "Русский Крест", собранный на базе Vite.

## Структура проекта

```
project/
├── dist/              # Собранная production версия
├── fonts/             # Шрифты проекта
├── img/              # Изображения и иконки
├── src/              # Исходный код
│   └── main.js       # Главная точка входа JavaScript
├── styles/           # CSS стили
│   └── main.css      # Основной файл стилей
├── index.html        # Главная HTML страница
├── package.json      # Зависимости проекта
└── vite.config.js    # Конфигурация Vite
```

## Установка

```bash
npm install
```

## Запуск для разработки

```bash
npm run dev
```

Проект будет доступен по адресу `http://localhost:3000`

## Сборка для production

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

## Предварительный просмотр production сборки

```bash
npm run preview
```

## Технологии

- **Vite** - быстрая сборка и разработка
- **Supabase** - база данных и бэкенд
- **Swiper** - слайдеры и карусели
- **Inputmask** - маски для полей ввода
- **CSS** - чистый CSS с переменными

## Особенности

- Адаптивная верстка для всех устройств
- Модульная архитектура JavaScript
- База данных новостей с Supabase
- Динамический пресс-центр с фильтрацией
- Детальные страницы статей
- Оптимизированная сборка ассетов
- Современные веб-стандарты

## Пресс-центр

Компонент пресс-центра включает:

- **Главная страница** - блок с последними 5 новостями
- **Список новостей** (`/news.html`) - страница со всеми новостями
- **Детальная страница** (`/article.html?slug=...`) - полная версия статьи
- **Фильтрация** - по категориям (Новости, СМИ о Фонде, События)
- **Пагинация** - кнопка "Показать ещё" для загрузки дополнительных статей
- **База данных** - все новости хранятся в Supabase

### Навигация

- Главная страница → новость в блоке "Пресс-центр" → детальная страница
- Главная страница → кнопка "Перейти в пресс-центр" → список новостей
- Список новостей → новость → детальная страница
- Меню → Пресс-центр → список новостей

### Структура базы данных

Таблица `news_articles`:
- `id` - уникальный идентификатор
- `title` - заголовок статьи
- `slug` - URL-friendly идентификатор
- `excerpt` - краткое описание
- `content` - полный текст статьи
- `image_url` - ссылка на изображение
- `category` - категория (news, media, events)
- `published_at` - дата публикации
- `is_featured` - отметка о важности
